# Image_processing_game

משימות להמשך:
1. לשפר קליטה של הצבע האדום (בשעת לילה פחות קלט אותו?)
2. להכניס את כל הטראשהולדים כתכונות של מריו
3. להכניס פיצ'ר עריכת הטראשהולדים דרך MAIN לפי לחיצה על מקש עריכה כלשהו
4. לנסות להבין את סד"פ להפעלת גראבים - זה לא סתם לחיצה כמו ימינה שמאלה במשחק, זה דורש איזשהו תיאום עם הקפיצה, טיימינג, ושלא ילחצו מקשים אחרים בזמן הזה ( אלא אם רוצים פעלול בכוונה)
5. לשייף ולדייק את זיהוי הגראב עוד יותר, ולהבהיק את הסימון שלו על המסך כל עוד לוחצים על המקש (כרגע זה רק כשזה מתקבל, ואנחנו לוחצים על המקש למשך 2 שניות החל מהרגע שזה התקבל)
6.  לאפיין סטופ ע"י טרנספורמציית האף (אחרי קפיצה) לזיהוי פיסוק ברגליים
7.  להכניס את הפיצ'ר של החלפת צד (גם זיהוי ע"י החלפת אדום ירוק לאחר קפיצה, וגם להוסיף את הלחיצה עצמה בPLAYER_CONTROL)
8.  ללטש
